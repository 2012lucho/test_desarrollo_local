# Router frontend

Centraliza las rutas de la aplicación.

- `/` → página de inicio (mensaje informativo)
- `/proyectos` → componente `abmProyectos.vue` (ABM de proyectos)

## Implementación

Archivo: `frontend/router/index.js`

Uso: `createRouter({ history: createWebHashHistory(), routes })`
